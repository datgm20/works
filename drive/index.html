教習所ドライビング

　教習所内を車で走り回るドライブシミュレーター。

動作環境

Windows10・・・

操作方法

キーボード

  • アクセル ... W
    • 急アクセル ... LShift + W
  • ブレーキ ... S
    • 急ブレーキ ... LShift + S
  • ハンドル ... A、D
  • シフト ... ↑、↓
  • ウィンカー ... ←、→
  • ハザード ... スペース
  • 視点切り替え ... R
  • 一人称視点変更
    • 左 ... Q
    • 右 ... E
  • やり直し ... LShift + R

ゲームパッド

  • アクセル ... Lスティック↑
    • 急アクセル ... Lスティック押し込み + Lスティック↑
  • ブレーキ ... Lスティック↓
    • 急ブレーキ ... Lスティック押し込み + Lスティック↓
  • ハンドル ... Lスティック←、→
  • シフト
    • 上 ... 2番
    • 下 ... 3番
  • ウィンカー ... 5番、6番
  • ハザード ... 4番
  • 視点切り替え ... 1番
  • Rスティック←、→

発進方法

ブレーキを踏みながらDレンジにする。(Sキーを押しながらDになるまで↓キーを押す)

使用アセット

  • 宇宙の背景: Solar System Scope. Solarsystemscope texture 8k stars milky way (クリエイティブコモンズ 表示 4.0 国際ライセンス)
  • BGM: shimtone. DOVA-SYNDROME. パステルアップ
